# SPEC-080: Narrative Structures

## Description

Replace 14 generic structures with narrative system: each structure generates procedural history. 5 new structure types: Library, Observatory, Camp, Castle Ruins, Archaeological Site.

## Requirements

### Structure Types (Enhanced + New)
1. **Village** (enhanced): layout around plaza, well + campfire + paths, variants (fishing/farming/mining/trading), 3-8 NPCs, village name + age + memorable event, loot chests
2. **Ancient Temple**: mossy stone, stairs, central altar, deity + ritual + curse lore, rare items + procedural book, traps (false floor, darts)
3. **Abandoned Mineshaft** (enhanced): tunnels with deteriorated wood supports, minecarts + rails + mineral chests, mining history, cave-in danger + hostile mobs
4. **Ruined Tower**: 3-5 partially destroyed floors, spiral staircase, panoramic view, builder + purpose lore, rare magic items
5. **Library** (NEW): bookshelves with procedural books, lost knowledge + sage lore, lore books + unique recipes, warm light + dust + silence
6. **Observatory** (NEW): dome with telescope, star maps, astronomer discovery lore, maps + navigation items, elevated platform with biome view
7. **Camp** (NEW): tents, campfire, supplies, explorers/merchants/bandits lore, 1-3 occupants (friendly or hostile), food + tools + maps
8. **Castle Ruins** (NEW): partially destroyed walls, keep tower, fallen kingdom + battle + hidden treasure lore, rare armor + enchanted weapons, hostile mobs + traps
9. **Archaeological Site** (NEW): partial excavation, fossils, artifacts, ancient civilization + discovery lore, unique artifacts + lore books, dust + silence + mystery
10. **Shipwreck** (enhanced): tilted partially buried hull, trade route + storm + treasure lore, chests + treasure maps, rotting wood + water + fish

### Procedural History
- Each structure gets: founding_event, decline_event, notable_figure
- History generated by LoreGenerator (SPEC-081)
- Books in structures reference the history

## Acceptance Criteria

- [ ] 5 new structure types generate correctly
- [ ] Each structure has procedural history (founding, decline, notable figure)
- [ ] Villages have variants (fishing/farming/mining/trading)
- [ ] Temples have traps
- [ ] Libraries have procedural books
- [ ] Castle Ruins have hostile mobs + traps
- [ ] Archaeological Sites have artifacts
- [ ] Loot matches structure type
- [ ] No console errors
